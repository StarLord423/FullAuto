# Migraine Daily Symptom Tracker

Thank you for using this printable.

This two-page printable is designed to help you record one migraine attack at a time. Fill in only what is useful. You do not need to complete every checkbox or line for the tracker to be helpful.

## How to Use

1. Print both pages for each migraine attack you want to record.
2. Use page 1 for the quick attack log: timing, pain level, symptoms, possible context, relief, and impact.
3. Use page 2 when you want extra room for medication/support details, pattern notes, and appointment questions.
4. Skip any section that is not useful today.
5. Bring several completed trackers to an appointment if you want to discuss patterns over time.

## A Note on Possible Triggers

Migraine patterns can be complicated. The "possible context" section is there to help you notice what was happening around an attack, not to blame yourself or prove one single cause.

## Printing Tips

- Print at 100% scale when possible.
- Black-and-white printing is fine.
- If the page prints too close to the edge, choose "Fit to printable area" in your printer settings.
- Use a clipboard, binder, or folder if you want to keep multiple pages together.

## Important Medical Note

This printable is a personal tracking and organization tool. It is not medical advice, diagnosis, or treatment. Always talk with a qualified healthcare professional about symptoms, medications, or care decisions.

If symptoms feel new, severe, sudden, or concerning, contact a qualified healthcare professional or emergency services.
